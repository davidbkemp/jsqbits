jsqbits runner

This is a crude JavaScript runner preloaded with the jsqbits library.

Try some quantum computation operations like: jsqbits('|001>').hadamard(ALL).rotateX(1, Math.PI/8).cnot(1,0)

Making the JavaScript shell run properly in different browsers was proving more effort than it is currently worth, and so it has been parked here. In the mean time, please feel free to use this crude JavaScript runner to try out jsqbits.
