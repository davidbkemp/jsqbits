jsqbits runner

An online quantum computer simulator

This is a JavaScript runner preloaded with the jsqbits library. Enter arbitrary JavaScript in the text area below and press the "Run" button. For more information, see the jsqbits user manual.

Try some quantum computation operations like: jsqbits('|001>').hadamard(jsqbits.ALL).rotateX(1, Math.PI/8).cnot(1,0)
